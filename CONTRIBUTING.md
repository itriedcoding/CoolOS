# CoolOS

A lightweight Linux distribution built for speed and simplicity.

## What is CoolOS?

CoolOS is a custom Linux distribution designed for:
- **Speed** - Fast boot times and low resource usage
- **Simplicity** - Clean, easy-to-use desktop environment
- **Completeness** - Pre-installed apps for everyday use
- **Flexibility** - Works on physical hardware and virtual machines

## Features

- Lightweight Openbox desktop environment
- Pre-installed web browsers (Firefox, Chromium)
- Office suite (LibreOffice)
- Media players (VLC, MPV)
- Development tools (Git, VS Code, Docker)
- Multimedia creation (GIMP, Audacity, Kdenlive)
- UEFI and BIOS boot support
- Virtual machine ready (QEMU, VirtualBox, VMware, Proxmox)

## Quick Start

```bash
# Clone
git clone https://github.com/itriedcoding/CoolOS.git
cd CoolOS

# Build
sudo ./scripts/build.sh

# Test
sudo ./scripts/test-qemu.sh
```
